**Copyright © 2026 Mucize Platform. All Rights Reserved.**

## Reporting Vulnerabilities

If you discover a security vulnerability in MIGE OS, please report it responsibly.

**Do not:**
- Create public issues
- Discuss the vulnerability publicly
- Exploit the vulnerability

**Do:**
- Email security@mucizeplatform.com
- Include detailed description
- Include steps to reproduce
- Allow us time to investigate and fix

## Supported Versions

Only the latest version is supported for security updates.

## Security Best Practices

- All secrets stored in GitHub Secrets
- No secrets in code
- No secrets in environment files
- Regular secret rotation
- Dependency scanning enabled
- CodeQL analysis enabled
